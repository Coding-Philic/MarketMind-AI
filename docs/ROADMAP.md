# MarketMind AI — MVP and Production Roadmap

## Phase 1 — Current MVP Foundation

- Keep the existing React + Express dashboard and agent simulation flow
- Preserve the current memory graph and hindsight ledger experience
- Continue using Gemini as the optional reasoning engine

## Phase 2 — Memory Persistence

- Add persistent storage for events, memories, and notes
- Move memory from browser-local simulation into a real backend store
- Introduce retrieval and indexing for historical intelligence

## Phase 3 — Knowledge Graph Expansion

- Connect company, sector, product, and event relationships
- Add graph traversal and similarity analysis
- Build causal explanation chains for revenue movement

## Phase 4 — Production Intelligence Stack

- Add FastAPI, PostgreSQL, Qdrant, Redis, Celery, Neo4j, and MinIO
- Introduce monitoring, logging, and cost tracking
- Expand the current demo into a research-grade autonomous pipeline

## Phase 5 — SaaS and Governance

- Add multi-tenant organization support
- Introduce RBAC, JWT, and audit trails
- Add usage limits, subscriptions, and admin controls

## Immediate Next Steps

1. Keep the current UI and simulation logic intact
2. Add backend persistence for memory and events
3. Add graph and vector storage for retrieval
4. Introduce a planning/orchestration layer for dynamic agent workflows
